# KadoTV Streaming Platform

Modern full-stack sports, movies, and series streaming app built with Next.js, Tailwind, Express, MongoDB, and JWT auth.

## Setup

1. Install dependencies
   ```bash
   npm run setup
   ```
2. Run the app
   ```bash
   npm run dev
   ```

## Structure

- `frontend/` — Next.js 15 app with responsive landing, login, dashboard, and player pages.
- `backend/` — Express API, MongoDB models, JWT authentication, admin routing, and media endpoints.
